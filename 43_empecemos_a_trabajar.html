4.3. Empecemos a trabajar.

Diccionario

LED

Definición:

Acrónimo de light-emitting diode (diodo emisor de luz). Es una fuente de luz de bajo consumo y muy duradera.

Ejemplo:

El teléfono móvil lleva una luz LED para iluminar.

Lenguaje de programación

Definición:

Es un lenguaje que permite realizar un programa para introducir instrucciones en una máquina informática.

Ejemplo:

Hay lenguajes de programación mediante bloques y escritos.

2. Veamos ahora un ejemplo de sensor analógico

¿Algo más de ayuda?

El montaje

En este ejemplo, "Entrada analógica" se demuestra la entrada analógica mediante la lectura de un sensor analógico en el pin analógico 0 y encender y apagar un diodo emisor de luz (LED) conectado al pin digital 13. La cantidad de tiempo el LED estará encendido y apagado dependiendo del valor obtenido por analogRead().

Como puedes comprobar en el circuito tenemos:

  * Potenciómetro conectado a la entrada analógica 0

  * pin central del potenciómetro al pin analógico

  * un pin lateral (cualquiera) a tierra

  * el pin del otro lado a +5V

  * El LED que encenderemos es el que viene incorporado a la placa conectado al pin 13 en el tablero.

El código

int sensorValue = 0;  //Define una variable con el valor inicial = 0

void setup()

{    //Define cómo serán los pines

  pinMode(A0, INPUT);    //Define el pin A0 como de entrada

  pinMode(LED_BUILTIN, OUTPUT);    //Define el pin del LED integrado como de salida

}

void loop()

{

   sensorValue = analogRead(A0);    // la variable creada toma el valor de la entrada del potenciómetro

   digitalWrite(LED_BUILTIN, HIGH);  // pone el LED en on

   delay(sensorValue);       // detendrá el programa el tiempo que indique <sensorValue> en milisegundos

   digitalWrite(LED_BUILTIN, LOW);    // pone el LED en off

  delay(sensorValue);     // detendrá el programa el tiempo que indique <sensorValue> en milisegundos

 }
